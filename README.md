1. **Установите зависимости:**
   ```bash
   composer install
   ```
2. **Подготовьте базу данных (Windows):**
   ```bash
   cp .env.example .env
   php artisan key:generate
   
   type nul > database/database.sqlite
   php artisan migrate:fresh --seeder=DemoSeeder
   ```

3. **Запустите сервер:**
   ```bash
   php artisan serve
   ```
---

## Тестирование в браузере

### 1. Менеджер (ID=1) — Видит всё
Должность "Manager" имеет доступ к категориям **Economy** и **Business**
Должны отобразиться **два автомобиля** (Lada и BMW), так как они оба свободны утром
[http://127.0.0.1:8000/api/cars/available?start_time=2025-12-31+09:00:00&end_time=2025-12-31+10:00:00](http://127.0.0.1:8000/api/cars/available?start_time=2025-12-31+09:00:00&end_time=2025-12-31+10:00:00)

### 2. Джуниор (ID=2) — Только Эконом
Должность "Junior" имеет доступ **только** к категории **Economy**
Даже если BMW свободен, Джуниор его **не увидит**
[http://127.0.0.1:8000/api/cars/available?start_time=2025-12-31+09:00:00&end_time=2025-12-31+10:00:00&user_id=2](http://127.0.0.1:8000/api/cars/available?start_time=2025-12-31+09:00:00&end_time=2025-12-31+10:00:00&user_id=2)

### 3. Проверка занятости (Для Менеджера)
Машина **BMW забронирована с 14:00 до 16:00**
Если запросить это время, в списке останется **только Lada**
[http://127.0.0.1:8000/api/cars/available?start_time=2025-12-31+15:00:00&end_time=2025-12-31+16:00:00](http://127.0.0.1:8000/api/cars/available?start_time=2025-12-31+15:00:00&end_time=2025-12-31+16:00:00)
